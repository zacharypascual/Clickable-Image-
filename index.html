Website Structure

Clickable Image that also shows the website structure with CSS

This project website is designed as an interactive and engaging platform for school purposes, combining creativity with basic web development skills. One of its main features is a clickable image that displays a random picture every time it is clicked. In addition, even when the user refreshes or restarts the website, a new random image still appears automatically. This makes the website more dynamic and enjoyable, encouraging users to explore and interact with the content instead of just viewing static elements.

The website also introduces a simple but effective structure using HTML and CSS. The layout includes a header, navigation bar, main content area, and footer, making the design organized and easy to follow. CSS is used to style the website by adding colors, spacing, borders, and alignment to different sections. It also enhances the appearance of the clickable image by applying effects such as hover styles, borders, and positioning, making the interface more visually appealing.

Overall, this project demonstrates how basic programming concepts can be used to create an interactive website. By combining the random image feature with a clean CSS layout, the website not only functions well but also looks presentable and user-friendly. It serves as a good example of how students can apply their knowledge of web design to build something both functional and creative.
